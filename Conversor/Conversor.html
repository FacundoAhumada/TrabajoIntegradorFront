Conversor de unidades

Calculadora digital

Este conversor de unidades te permite realizar conversiones entre diferentes sistemas de medida. A continuación te mostramos las opciones disponibles:

Opción 1: Temperatura

Para convertir entre diferentes escalas de temperatura, puedes elegir entre las siguientes unidades:

  • Celsius
  • Kelvin
  • Fahrenheit

Opción 2: Peso

Para convertir entre diferentes unidades de peso, las opciones disponibles son:

  • Gramos
  • Kilogramos
  • Libras

Opción 3: Longitud

Para convertir entre distintas unidades de longitud, las opciones incluyen:

  • Metros
  • Kilómetros
  • Millas
  • Pies
